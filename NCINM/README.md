## Version History

### **2024-12-15 — Official Release (Version 2.0.20241215)**
- Links to the user manual and user forum added
- Bug fix: Radionuclide selection not matching radiopharmaceutical selection

### **2024-01-24 — Official Release (Version 2.0.20240124)**
- Biokinetic models developed for pediatric thyroid cancer patients  
  (Kwon et al., *Journal of Radiological Protection*, 2023) added
- Biokinetic models developed for adult thyroid cancer patients  
  (Kwon et al., *Journal of Radiological Protection*, in press) added
- Biokinetic models derived from older ICRP Publications 53, 80, and 106 removed when
  updated data from **ICRP Publication 128** were available

### 2022-12-15
- Mismatch between radionuclides and radiopharmaceuticals corrected
- Radiopharmaceutical names sorted alphabetically
- Effective dose calculation algorithm revised

### **2022-12-14 — Official Release (Version 2.0.20221214)**
- NCINM 2.0 published:  
  Villoing et al., *“Organ dose calculator for diagnostic nuclear medicine patients based on the ICRP reference voxel phantoms and biokinetic models,”*  
  **Biomedical Physics & Engineering Express**, 9:015004 (2023)
- For ICRP pediatric and adult phantoms, when blood is the source region, target organ
  mass now includes blood mass, consistent with **ICRP Publication 133**
- Biokinetic data extended to **230 radiopharmaceuticals**  
  (ICRP Publications 53, 80, 106, and 128)

### 2022-11-09
- Issues related to mismatches between radionuclides and radiopharmaceuticals fixed
- Copy-and-paste via mouse dragging enabled

### 2022-09-19
- Self- and cross-fire SAFs for the gastrointestinal tract in adults updated using
  **ICRP Publication 133**
- Bug preventing export of S values for ICRP phantoms fixed

### 2022-05-12
- Biokinetic data for four additional radiopharmaceuticals added
- Issues with the ovaries in the ICRP 15-year-old female phantom corrected

### 2022-04-20
- NCI phantom–based active marrow and endosteum SAFs and S values revised using the
  latest dose response functions adopted by **ICRP Committee 2**

### 2022-03-29
- Logical errors in skeletal dose calculations corrected
- Effective dose using ICRP Publication 60 tissue weighting factors removed
- Effective dose using ICRP Publication 103 tissue weighting factors incorporated into
  target organ doses automatically copied to the clipboard
- Errors in blood-inclusive target organ mass values corrected
- Biokinetic data extended to **101 radiopharmaceuticals**
  (ICRP Publications 53, 80, and 106)

### 2021-11-29
- Frontal images of ICRP voxel phantoms implemented, replacing frontal images of NCI
  hybrid phantoms

### 2021-11-13
- Zero adipose mass issue for the ICRP newborn female phantom corrected
- Errors in the Windows installation file fixed
- User interface revised to be more compact

### 2021-10-19
- Biokinetic data for a total of **62 radiopharmaceuticals** made available
- User Manual and User Forum menu items added under the **Help** menu

### 2021-05-15
- Blood mass added to target organ mass in ICRP pediatric reference phantoms, consistent
  with adult ICRP phantoms in **ICRP Publication 133**

### **2021-05-13 — Official Release (Version 2.0.20210301)**
- Biokinetic models for 12 radiopharmaceuticals extracted from multiple ICRP publications
- S values from the 12 ICRP reference pediatric and adult phantoms added

### 2020-07-20
- NCINM 1.0 published:  
  Villoing et al., *“NCINM: organ dose calculator for patients undergoing nuclear medicine procedures,”*  
  **Biomedical Physics & Engineering Express**, 6:055010 (2020)

### **2020-03-12 — Official Release**
- Comprehensive benchmarking conducted against **OLINDA/EXM 1.0** and **IDAC 2.1**

### 2019-08-22
- NCINM 1.0 created based on the 12 NCI reference phantoms
- Alpha testing initiated
